Проект больше не про «еще один предсказатель ГЭБ». Ядро - метод, который снижает ошибки интерпретации: первичный прогноз по структуре вещества остается входной точкой, а новые данные меняют вывод только тогда, когда они биологически совместимы с конкретным блоком решения. hERG, DILI или общая токсичность не должны напрямую переворачивать вывод о поступлении в мозг, если у них нет объясненного пути через переносчики, повреждение барьера, свободную фракцию или другой заранее описанный механизм.

// flagship project
GEB-Contour / ГЭБ-Контур error-reduction method · evidence routing · CLI first
Метод контекстно-совместимого обновления данных для оценки проницаемости ГЭБ
  • Научная новизна: не само предсказание logBB, не общая ADMET-панель и не краткий пересказ. Новое ядро - формальный контур, где каждый опыт имеет паспорт, относится к конкретному биологическому блоку и может обновлять только совместимые выводы.
  • Вход: SMILES, первичный прогноз In Silico BBB, паспорт источника, тип опыта, единицы, проверенность, блок совместимости: проницаемость, переносчики, связывание, токсикология, средовой контекст, итог по мозгу.
  • Цикл: первичная вероятность → проверка совместимости → ограниченное обновление → разбор расхождений → рекомендация следующего опыта → новый паспорт данных. Байесовское обновление используется как часть цикла, а не как отдельная главная идея.
  • Токсикология: учитывается только как путь, который может изменить транспорт или состояние барьера. Системный яд сам по себе не равен проникновению в мозг; он становится значимым только через объясненный механизм.
  • Практическая ценность: контур не обязан выигрывать у сильной ML-модели на каждой простой молекуле. Его задача - уменьшать ложные обновления, показывать, где данных недостаточно, и предлагать следующий опыт.
Context-compatible evidence updating for blood-brain barrier assessment
  • GEB-Contour is an error-reduction method for BBB permeability assessment, not a standalone BBB predictor.
  • The system starts from a primary In Silico BBB/logBB estimate, attaches a data passport to every observation, maps each assay to a biological block, and allows updates only along compatible paths.
  • The loop is: prior estimate → compatibility check → constrained update → discordance resolution → next-assay recommendation → revised evidence state.
  • Toxicology is included only when it has a biologically defined route to BBB transport, barrier integrity, free fraction, or transporter behavior.
  • The measurable value is fewer false updates and better abstention/explanation on complex molecules, especially outside the comfortable domain of the primary model.

P-gp, BCRP, LAT1, efflux ratio, ингибирование/субстратность. Не подменяет Kp,uu без связи с экспозицией.

Системная токсикология
hERG, DILI, тканевое повреждение, воспаление. Меняет вывод о ГЭБ только при описанном пути к барьеру, транспорту или экспозиции.
Среда / кишечник / микробиота
Не центральное ядро. Может быть контекстом, если влияет на переносчики, метаболизм, воспаление или системную экспозицию.
